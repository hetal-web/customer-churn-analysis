# Customer Churn Analysis

## Objective
Analyze customer data to understand churn patterns and retention behavior.

## Tools Used
- Python
- Pandas
- Seaborn

## Key Analysis
- Churn rate calculation
- Impact of contract type and tenure
- Customer behavior trends

## Insights
- Monthly contract users have higher churn
- Low tenure customers are more likely to leave

## Conclusion
This project helps identify at-risk customers and supports better retention strategies.
